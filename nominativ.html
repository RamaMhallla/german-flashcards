اختبار Nominativ

اختبار تفاعلي: Der Nominativ

اختبر فهمك لحالة الرفع بالألمانية باستخدام أمثلة الدفتر (sein, werden, bleiben, heißen, scheinen) + أسئلة Wer/Was والردود بـ kein/e.
النتيجة: 0 / 0
سؤال 1 / ?
تلميح: الأسماء بعد sein / werden / bleiben / heißen / scheinen تكون في Nominativ. —
📒 أمثلة الدفتر المُستخدمة (انقر للعرض)
Das ist der Mann / ein Mann / kein Mann
Das ist die Frau / eine Frau / keine Frau
Das ist das Kind / ein Kind / kein Kind
Das sind die Kinder / Kinder / keine Kinder
Wer telefoniert? — Der Mann telefoniert.
Was ist klein? — Das Auto ist klein.
Wer ist das? — Das ist Sarah.
Ist das eine Frau? — Nein, das ist ein Mann. Das ist keine Frau.
Ist das ein Auto? — Nein, das ist kein Auto. Das ist ein Teller.
Sind das Bananen? — Nein، das sind keine Bananen.
Ist das ein Messer? — Nein، das ist kein Messer. Das ist eine Gabel.
Sie ist (die) Lehrerin. / Sie will eine Lehrerin werden.
Er bleibt mein Freund. · Sie heißt Carolina. · Er scheint ein guter Chef zu sein.
